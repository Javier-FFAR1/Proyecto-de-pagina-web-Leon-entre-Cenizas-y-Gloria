Patrimonio Cultural de León

Este proyecto nace con el objetivo de preservar, difundir y celebrar la rica historia, los monumentos y las tradiciones de la ciudad de León, Nicaragua. Un espacio digital para que locales y visitantes descubran la grandeza cultural de la "Capital Cultural de Nicaragua".

Objetivo

Preservar la memoria histórica de León, difundir sus monumentos más representativos y fomentar el turismo cultural mediante una plataforma accesible, moderna y educativa.

Tecnologías utilizadas

HTML5 CSS3 JavaScript Node.js Express.js MySQL mysql2 REST API Fetch API Font Awesome Responsive Design

Fuentes de información

Alcaldía de León, Instituto de Historia de Nicaragua, UNESCO (Patrimonio de la Humanidad), archivos históricos locales y testimonios de la comunidad leonesa.
